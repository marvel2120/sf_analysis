# 🚀 智能投资分析系统

基于 Streamlit 的股票和基金分析Web应用，提供专业的技术分析和投资建议。

## 📋 功能特性

### 📊 股票分析
- **技术分析**: 趋势识别、相对强弱评估
- **突破检测**: 自动识别价格突破信号
- **量能分析**: 成交量配合度评估
- **投资建议**: 基于多重指标的买卖建议

### 💰 基金分析
- **业绩评估**: 多时间段收益分析
- **风险分析**: 波动率、最大回撤、夏普比率
- **基准对比**: 与沪深300指数对比
- **综合评级**: 趋势+业绩+风险的综合评分

## 🛠️ 技术栈

- **后端**: Python 3.8+
- **数据获取**: AkShare (实时金融数据)
- **分析引擎**: Pandas, NumPy, SciPy
- **前端**: Streamlit (现代化Web界面)
- **可视化**: Plotly (交互式图表)


### 2. 创建虚拟环境 (推荐)


```bash

```

## 📖 使用方法

### 股票分析
1. 在左侧选择"股票分析"
2. 输入股票代码（如：000001、600000、300001）
3. 点击"开始分析"按钮
4. 查看技术分析报告和投资建议

### 基金分析
1. 在左侧选择"基金分析"
2. 输入基金代码（如：000001、110011）
3. 点击"开始分析"按钮
4. 查看基金业绩分析和综合评级

## 📊 分析指标说明

### 股票指标
- **趋势阶段**: 上升、下降、震荡
- **相对强弱**: 相对于沪深300的表现（>1表示跑赢）
- **突破信号**: 价格是否突破关键阻力位
- **量能配合**: 成交量是否支持价格走势

### 基金指标
- **收益指标**: 近1年、2年、3年收益率
- **风险指标**: 年化波动率、最大回撤
- **风险调整后收益**: 夏普比率
- **相对强弱**: 与沪深300对比的相对表现

## 🔧 故障排除

### 常见问题

1. **数据获取失败**
   - 检查网络连接
   - 确认代码格式正确
   - 稍后重试（可能是数据源暂时不可用）

2. **分析超时**
   - 减少分析的时间范围
   - 检查系统资源使用情况

3. **依赖安装失败**
   - 确保Python版本 >= 3.8
   - 尝试升级pip: `pip install --upgrade pip`
